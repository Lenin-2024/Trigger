# О проекте

Этот проект предоставляет современный графический интерфейс терминала, который запускает и взаимодействует с TinyEMU - эмулятором RISC-V архитектуры. Интерфейс обеспечивает плавный ввод/вывод команд через интуитивно понятный графический интерфейс.

**Основные возможности:**

* **Эмуляция RISC-V** - Полная интеграция с TinyEMU для эмуляции системы RISC-V
* **Межпроцессное взаимодействие** - Использование UNIX pipes для эффективной коммуникации
* **Обработка ANSI escape последовательностей** - Корректная обработка управляющих последовательностей терминала
* **Интерактивный ввод** - Ввод команд напрямую в графическом интерфейсе

**Технические особенности**
* **Реальное время** - Мгновенный отклик на пользовательский ввод
* **Буферизация вывода** - Умное управление буфером для больших объемов данных
* **Очистка вывода** - Фильтрация служебных символов и escape-последовательностей
* **Гибкая архитектура** - Легко расширяемая кодовая база

# Требования

**Системные зависимости:**
* Компилятор C (gcc/clang)
* Библиотека Raylib
* TinyEMU эмулятор

**Обязательные зависимости:**
* sudo apt update
* sudo apt install build-essential git make
* sudo apt install libasound2-dev libx11-dev libxrandr-dev libxi-dev libgl1-mesa-dev libglu1-mesa-dev libxcursor-dev libxinerama-dev libwayland-dev libxkbcommon-dev

# Сборка проекта
1. Для сборки проекта скчайте его использовав команду git clone https://github.com/Lenin-2024/Trigger или же Скачайте его как архив и расспакуйте.
2. Далее перейдите в папку с проектом, откройте тиерминал и используйте команду "make" в терминале.
3. Далее ожидайте завершение сборки проекта. По завершению у вас появится файл "main" - это ти есть испольняемый файл. Теперь вы можете запускать данную программу
